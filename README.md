# Closhot-Skills · 业务专属技能库

Closhot项目专属的Claude Code Skill库，覆盖所有业务逻辑。

## 使用方式
电脑到了，告诉Claude Code：
「读Claude-Skills和Closhot-Skills里的所有Skill，按照规范开发Closhot项目」

## Skill列表（Claude Code自动生成）
等电脑到了，运行以下命令让Claude Code生成所有Skill：

「帮我建一个Closhot专属Skill库，覆盖以下所有业务方向：

产品设计原则：
- 先价值后利益/简单不复杂/不打扰原则
- 渐进式功能披露
- UI高级感规范（黑白灰/多留白/动画克制）
- 错误提示语规范（站用户角度说清影响）

AI Router调度系统：
- 极速预览 → Cloudflare Workers AI（免费）
- 普通生成 → 硅基流动Flux
- 高级生成 → OpenRouter Flux官方
- 模特换装 → 硅基流动IDM-VTON
- 文案生成 → OpenRouter DeepSeek
- 图片分析 → OpenRouter Gemini
- 去背景 → 硅基流动BRIA RMBG
- 模型热切换规范

用量系统：
- 成功才扣额度，失败不扣
- 免费版/Starter/Pro/企业版额度规则
- 高级生成和普通生成分开计算
- 软删除规则

支付系统：
- 虎皮椒支付接入规范
- 订阅制/按月付费规则
- 支付成功后权益变更规则

水印系统：
- 免费用户有水印
- 分享可免费去水印（本次）
- 付费用户永久无水印
- 水印精美设计规范

智能分析系统：
- 三层递进规范（评分→大方向→完整分析）
- 对生成结果只说「可以更好的机会」不说「问题」
- 各平台独立评分规范（淘宝/拼多多/抖音/小红书）
- 分析结果客观不吹捧

提示词系统：
- 各品类提示词模板（连衣裙/外套/T恤/裤子/童装/男装）
- 负向提示词库
- 固定种子值规范
- 中文描述转英文提示词规范
- 提示词版本管理规范

商品保真系统：
- 花纹/Logo/面料/版型不变形规范
- IP-Adapter使用规范
- ControlNet使用规范
- 后处理流程（锐化/色彩校正/裁剪）

图片质量系统：
- 质量评分阈值规范
- 自动重试规则
- 降低AI感规范（手部/人脸/边缘）

用户体验规范：
- 未登录可试用1次
- 新手引导问卷（3题30秒）
- 注册礼3次高级生成
- 生成前先选平台
- 双模式设计（一键选风格/对话框）

GEO/SEO规范：
- 用户真实搜索词覆盖
- FAQ页面规范
- JSON-LD结构化数据规范
- 知乎/小红书内容规范

冷启动规范：
- AI数字人短视频制作规范
- Before/After对比内容规范
- 种子用户运营规范

每个Skill都要：
- 详细可执行的规则
- 示例代码
- 注意事项」
