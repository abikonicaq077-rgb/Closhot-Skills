# Mainshot-Skills · 业务专属技能库

Mainshot项目专属的Claude Code Skill库，覆盖所有业务逻辑。

## 使用方式
电脑到了，告诉Claude Code：
「读Claude-Skills和Mainshot-Skills里的所有Skill，按照规范开发Mainshot项目」
